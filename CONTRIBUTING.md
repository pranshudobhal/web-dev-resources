# Contributing

If you're reading this, you're awesome! ❤️

Thank you for helping us make this project great. Here are a few guidelines that will help you along the way.

## Guidelines

- Before committing to this repository, please check if the resource has already been listed or not.

- Add a single resource in one pull request as it will be easier to review.

- Pull request title should include resource name and category.
  - Example: `[Font Awesome Icons] -> [Icons]`

- Pull request description should include a link to the resource and a brief description.

- If appropriate category is not present, create a new one and mention it in pull request description.

- The resource must be completely free or have a free tier.

Thank you! ❤️
